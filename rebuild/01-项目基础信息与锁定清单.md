## 1.1 项目基本信息

| 项目 | 值 |

| 当前版本 | 1.0.0 |

## 1.2 技术栈锁定清单

以下资产**不可修改**，任何重构必须保持兼容：

| 锁定项 | 说明 |
|---|---|
| Supabase 项目 ID | `zvemahqskgluhirzbcqu`，所有表、Edge Function、RLS 策略均绑定此项目 |

## 1.7 鉴权架构

```
┌─────────────────────────────────────────────────────────┐
│                    main.jsx                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │ AuthProvider                                     │    │
│  │  • supabase.auth.getSession() → 初始session      │    │
│  │  • onAuthStateChange → 监听登录/登出              │    │
│  │  • signOut() → 清除session                        │    │
│  │  → AuthContext { user, session, loading, signOut }│    │
│  └─────────────────────────────────────────────────┘    │
│                         ↓                                │
│  ┌─────────────────────────────────────────────────┐    │
│  │ AppProvider (context/AppContext.jsx)              │    │
│  │  • userId = supaUser?.id (UUID)                   │    │
│  │  • isLoggedIn = !!session                         │    │
│  │  • syncUserStatsOnLogin(userId)                   │    │
│  │  • loadDictFromDB() → 分词词典                    │    │
│  │  → AppContext { auth, nav, theme, handlers }      │    │
│  └─────────────────────────────────────────────────┘    │
│                         ↓                                │
│  ┌─────────────────────────────────────────────────┐    │
│  │ App.jsx                                          │    │
│  │  • if (!isLoggedIn) → <LoginPage />               │    │
│  │  • if (isLoggedIn) → 4-tab主界面 + overlays       │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

**登录方式（7种）：**
1. 邮箱 + 密码（`signInWithEmail`）
2. 邮箱 + 密码注册（`signUpWithEmail`）
3. Google OAuth（`signInWithOAuth('google')`）
4. GitHub OAuth（`signInWithOAuth('github')`）
5. Magic Link 邮件登录（`sendMagicLink`）
6. Brevo OTP 登录验证码（`sendOtp` + `verifyBrevoOtp`）
7. Brevo OTP 重置密码（3步流程：发送→验证→设新密码）

## 1.8 导航架构

**无 URL 路由**——纯状态驱动页面切换：

```
底部4个Tab：
  home (首页) → HomePage
  words (单词本) → WordBookPage
  learn (学习) → LearnPage
  me (我的) → ProfilePage

3层Overlay（绝对定位覆盖）：
  UnknownWordPage → AI生成词条 → 转为 WordDetailPage
  WordDetailPage → 词条详情（多义项、收藏、近反义词）
  SentenceDetail → 句子详情（逐词分词分析）

导航栈：
  navStack[] → 后退历史
  navForward[] → 前进历史
  visitedPages Set → keep-alive（已访问页面不销毁）
  resetNav() → 切换Tab时清空overlay和导航栈
```
