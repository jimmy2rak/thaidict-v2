## 5.1 首页（HomePage.jsx）

### 页面结构
```
┌─────────────────────────────────┐
│ 统计栏：连续打卡X天 | 词典X词     │
├─────────────────────────────────┤
│ [🔍 搜索输入框（400ms防抖）]      │
├─────────────────────────────────┤
│ 搜索结果列表（有query时）          │
│  或                              │
│ 每日一词卡片 + 每日一句卡片        │
│  （无query时）                    │
├─────────────────────────────────┤
│ 本月打卡日历                      │
└─────────────────────────────────┘
```

### 交互元素

**搜索输入框**
- 占位文本：「搜索泰语单词、中文…」
- 输入时右侧显示清除按钮（×）
- 400ms 防抖后触发 `searchWords(query)`
- 搜索中显示 spinner
- 搜索来源：dictionary_full（精确+模糊）+ search_words_zh RPC + community_words
- 结果按优先级排序：精确匹配 > enriched > sense_count 降序 > 社区词

**搜索结果列表项**
- 显示：泰语词（粗体）、罗马拼音、词性标签、中文释义
- 点击 → `handleWordTap(word)` → 查字典 → 有数据进 WordDetailPage，无数据进 UnknownWordPage

**每日一词卡片**
- 数据来源：`loadDailyPick()` → `daily_picks` 表 → JOIN `dictionary_full`
- 显示：泰语词、罗马拼音、词性、中文释义、例句
- 右侧：TTS 朗读按钮 + 刷新按钮
- 点击卡片 → `handleWordTap` 进入词条详情
- 刷新按钮 → `refreshDailyPick('word')` → 随机获取新词

**每日一句卡片**
- 数据来源：`loadDailyPick()` → `daily_picks` 表 → JOIN `sentences`
- 显示：泰语句子、中文翻译、分词结果（enrichSegmented 补充含义）
- 分词结果中每个词可点击 → `handleWordTap`
- 点击句子卡片 → `setSelectedSentence` → 进入 SentenceDetail overlay
- 刷新按钮 → `refreshDailyPick('sentence')`

**统计栏**
- 连续打卡天数：`getStreak(userId)` → `user_learning_progress.streak_days`
- 词典总数：`getDictionaryCount()` → `dictionary` 表 count（⚠️ Bug B-3）
- 收藏数：`getBookmarks(userId).length`
- 本月打卡天数：`getMonthlyCheckinStreak(userId)`

## 5.2 单词本（WordBookPage.jsx）

### 页面结构——4个内部Tab
```
┌─────────────────────────────────┐
│ [最近] [单词夹] [句子夹] [单词书] │
├─────────────────────────────────┤
│ Tab内容区域                      │
└─────────────────────────────────┘
```

### Tab 1：最近查词
- 数据来源：`getUserRecentWords(userId, 50)`
- 显示最近查看的词列表（带字典数据富化）
- 点击 → `handleWordTap`

### Tab 2：单词夹
- 数据来源：`getFolders(userId)` → `folder_type = 'word'`
- 文件夹列表：名称、颜色、词数
- 操作：
  - 点击文件夹 → 展开显示内部单词列表（`getFolderWords(folderId)`）
  - 长按/滑动 → 重命名、删除
  - 「+ 新建文件夹」按钮 → `createFolder(userId, name, color, 'word')`
  - 单词卡片上 「移除」按钮 → `removeWordFromFolder(folderId, word)`
  - 从词条详情页「添加到文件夹」→ 弹出文件夹选择器

### Tab 3：句子夹
- 数据来源：`getFolders(userId)` → `folder_type = 'sentence'`
- 操作类似单词夹，但存储的是句子收藏
- 句子来源：`getFolderSentences(folderId)` → JOIN `sentences` 表
- 点击句子 → `setSelectedSentence` → SentenceDetail overlay

### Tab 4：单词书
- 当前为空状态（见文档3 3.8节）

## 5.3 学习（LearnPage.jsx）

### 页面结构
```
┌─────────────────────────────────┐
│ 打卡任务区域                      │
│  □ 背单词 15分钟   [打卡]         │
│  ☑ 阅读练习 20分钟 [已完成]       │
│  [+ 添加任务] [AI生成计划]        │
├─────────────────────────────────┤
│ 快捷入口                          │
│  [笔记] [短语] [统计] [调整计划]   │
└─────────────────────────────────┘
```

### 打卡任务区域
- 数据来源：`getCheckinTasks(userId)` → 按 `sort_order` 排序
- 每个任务显示：任务名、时长、打卡状态
- 打卡状态：`getCheckinCompletions(userId, todayCST)` 判断今日是否完成
- 点击打卡 → `toggleCheckinTaskCompletion(userId, taskId, todayCST, !completed)`
  - 完成时自动增加 `study_minutes`（⚠️ Bug B-6 竞态）
  - 取消时自动减少 `study_minutes`
- 「+ 添加任务」→ 弹出任务创建表单（类型、名称、时长、计划日）
- 「AI生成计划」→ 进入 `AdjustPlanSection`

### 快捷入口
- **笔记** → `getNotes(userId)` → 笔记列表 → 点击进入 `NoteEditorSection` 或 `NotesDetailSection`
- **短语** → 进入 `PhrasesSection`（idioms/buddhist/daily 三类句子）
- **统计** → 进入 `StatsSection`（热力图 + 柱状图）
- **调整计划** → 进入 `AdjustPlanSection`（AI生成学习计划 + 任务CRUD）

### 头像上传
- 点击头像 → 选择图片文件
- 上传到 Supabase Storage `user-assets` bucket
- 路径：`avatars/{userId}/{timestamp}.{ext}`
- 更新用户 metadata：`updateUserProfile({ avatar_url: publicUrl })`

### API 密钥管理
- 数据来源：`getApiKeys(userId)` → `user_api_keys` 表
- 默认API：`getDefaultApi(userId)` → `user_settings.default_api_id`
- 添加密钥：provider、apiKey、base_url、model、name
- 删除密钥：`deleteApiKey(keyId)`
- 设为默认：`setDefaultApi(userId, apiId)`

### WebDAV 配置
- 存储在 `user_settings` 表（webdav_url, webdav_user, webdav_pass_enc）
- ⚠️ 密码加密逻辑缺失（见文档3 3.1节）

### 主题切换
- `colorMode` 存储在 `localStorage('thaidict-color-mode')` 和 `AppContext`
- 选项：`light` / `dark` / `system`
- 切换时设置 `document.documentElement.dataset.theme`

## 5.5 登录页（LoginPage.jsx）

### 页面结构
```
┌─────────────────────────────────┐
│ Logo + 应用名称                   │
├─────────────────────────────────┤
│ 邮箱输入框                        │
│ 密码输入框                        │
│ [登录] [注册]                     │
├─────────────────────────────────┤
│ ─── 其他登录方式 ───              │
│ [Google 登录] [GitHub 登录]       │
│ [Magic Link 登录]                │
├─────────────────────────────────┤
│ OTP 验证区域（条件显示）           │
│  6位验证码输入框                   │
│  [发送验证码] [验证]              │
├─────────────────────────────────┤
│ [忘记密码?]                      │
└─────────────────────────────────┘
```

### 登录流程
1. **邮箱密码**：`signInWithEmail(email, password)` → 成功则 AuthContext 自动更新
2. **注册**：`signUpWithEmail(email, password, username)` → ⚠️ 需确认邮箱（Bug D-2）
3. **OAuth**：`signInWithOAuth('google' | 'github')` → 重定向到 OAuth → 回调 `/auth/callback`（⚠️ Bug D-1）
4. **Magic Link**：`sendMagicLink(email)` → 邮箱点击链接 → 自动登录
5. **OTP 登录**：`sendOtp(email, 'login')` → 输入6位码 → `verifyBrevoOtp(email, code, 'login')` → 成功后自动 `signInWithEmail`

### 忘记密码导航
- 点击「忘记密码?」→ `onNavigate('reset-password')` → 切换到 `ResetPasswordPage`

## 5.6 重置密码页（ResetPasswordPage.jsx）

### 3步流程
```
Step 1: 输入邮箱 → 发送重置验证码
Step 2: 输入6位验证码 → 验证
Step 3: 输入新密码 → 更新密码
```

1. `sendOtp(email, 'reset')` → Brevo 发送重置验证码
2. `verifyBrevoOtp(email, code, 'reset')` → 验证成功返回 email
3. `supabase.auth.updateUser({ password: newPassword })` → 更新密码
4. 自动登录 → 返回主界面

## 5.8 未知词条 Overlay（UnknownWordPage.jsx）

### 页面结构
```
┌─────────────────────────────────┐
│ 未知词条：_unknown_word_          │
├─────────────────────────────────┤
│ 词典中未找到此词                  │
│ 中文提示（可选）：[输入框]         │
│ [AI 生成词条]                    │
├─────────────────────────────────┤
│ 生成中... spinner                │
│ 或                               │
│ 生成结果预览                      │
└─────────────────────────────────┘
```

### AI 生成流程
1. 用户点击「AI 生成词条」
2. `handleGenerated(word, zhHint)` 被调用
3. 检查用户是否有默认 API 密钥 → 有则使用用户密钥，无则使用系统密钥
4. 构造 prompt → `callAiProxy(prompt, userApi)` → Edge Function 处理
5. 解析返回的 JSON → 构造 wordEntry → 存入 `generatedWords`
6. 同时 `saveCommunityWord(wordEntry, userId, zhHint)` → 存入 `community_words`
7. 自动切换到 `WordDetailPage` 显示生成结果

## 5.10 Subsection 页面

### 5.10.1 AI学习计划（AdjustPlanSection.jsx — 1489行）

**功能：**
1. AI 生成个性化学习计划
   - 输入：学习水平（零基础/初级/中级/高级）、每日可用时间
   - 调用 `callAiProxy` 生成 3-5 个学习任务
   - 每个任务包含：类型、名称、时长、计划日
2. 任务 CRUD
   - 创建：`createCheckinTask(userId, taskData)`
   - 编辑：`updateCheckinTask(taskId, updates)`
   - 删除（软删除）：`deleteCheckinTask(taskId)` → 设 `is_active = false`
3. 任务类型：单词、语法、阅读、听力、口语、写作、自定义
4. 时长选项：5/10/15/20/30/45/60 分钟
5. 计划日：周一到周日，可多选

### 5.10.2 句子列表（PhrasesSection.jsx）

**功能：**
- 三类句子：idioms（成语）、buddhist（佛学）、daily（日常）
- 数据来源：`getSentencesByCategory(category)`
- 也包含硬编码的 `phraseData.js` 数据
- 点击句子 → `setSelectedSentence` → SentenceDetail overlay

### 5.10.3 学习统计（StatsSection.jsx）

**功能：**
- 打卡热力图：`getCheckinHeatmapData(userId, 35)` → 最近5周每日打卡次数
- 本周学习时长：`getWeeklyStudyMinutes(userId)` → 周一到周日柱状图
- 使用 Recharts 的 `BarChart` 和自定义热力图组件

### 5.10.4 笔记编辑器（NoteEditorSection.jsx）

**功能：**
- 创建笔记：`createNote(userId, title, content, color)`
- 编辑笔记：`updateNote(noteId, { title, content, color })`
- 删除笔记：`deleteNote(noteId)`
- 支持 Markdown 格式（简单文本编辑）
- 颜色选择：预设 6 种颜色

### 5.10.5 提醒设置（ReminderPage.jsx）

**功能：**
- 提醒开关：`saveUserSettings(userId, { reminder_enabled: true/false })`
- 提醒时间设置
- 发送测试提醒：`sendReminder({ email, tasks, time })`
- 任务列表显示（从 checkin_tasks 获取）
