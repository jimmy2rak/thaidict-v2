## 词笺（ThaiDict）&mdash; 中泰双语智能词典

**词笺** 是一款面向中泰双语学习者的智能词典应用，覆盖 **62,000+ 泰语词汇**，支持全文检索、例句发音、短语库、AI 辅助学习、WebDAV 云端备份等功能。

### 功能特性

- **词典检索** &mdash; 支持泰语、中文、拼音多维度搜索，实时分词高亮
- **每日推荐** &mdash; 每日一词 + 每日一句，碎片化学习
- **短语库** &mdash; 按主题（问候、餐饮、交通等）浏览常用泰语短语
- **AI 学习辅助** &mdash; AI 生成词条，提交审批后入库；支持自定义 API Key
- **学习打卡** &mdash; 连续打卡统计、学习热力图、成就系统
- **单词本** &mdash; 已学/收藏词条管理
- **导出与备份** &mdash; 支持 JSON / Markdown / Word 多格式导出，WebDAV 加密上传
- **多平台登录** &mdash; 邮箱密码 / Google / GitHub OAuth / 验证码登录
- **响应式** &mdash; 移动端优先设计，适配 iOS / Android PWA

### 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 14 (App Router) + React 18 |
| 样式 | CSS Variables + 新中式奶油风格 |
| 图标 | lucide-react |
| 图表 | recharts |
| 后端 | Supabase (PostgreSQL + Auth + RLS + Edge Functions) |
| 部署 | Vercel (自动 CI/CD) |
| 分词 | 基于词典的最大正向匹配 |
| 邮件 | Brevo (Sendinblue) 事务邮件 |

## ThaiDict &mdash; A Chinese-Thai Bilingual Intelligent Dictionary

**ThaiDict** is an intelligent dictionary application for Chinese-Thai bilingual learners, covering **62,000+ Thai words** with full-text search, sentence pronunciation, phrase library, AI-assisted learning, and WebDAV cloud backup.

### Features

- **Dictionary Search** &mdash; Multi-dimensional search in Thai, Chinese, and romanization with real-time word segmentation
- **Daily Picks** &mdash; Word of the Day + Sentence of the Day for micro-learning
- **Phrase Library** &mdash; Browse common Thai phrases by category (greetings, dining, transport, etc.)
- **AI Learning Assistant** &mdash; AI-generated entries submitted for admin approval; custom API key support
- **Study Check-ins** &mdash; Streak tracking, heatmap, and achievement system
- **Word Book** &mdash; Manage learned and bookmarked words
- **Export &amp; Backup** &mdash; Export in JSON / Markdown / Word formats; encrypted WebDAV upload
- **Multi-platform Login** &mdash; Email+Password / Google / GitHub OAuth / OTP verification
- **Responsive** &mdash; Mobile-first design, compatible with iOS / Android PWA

### Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 14 (App Router) + React 18 |
| Styling | CSS Variables + Neo-Chinese Cream aesthetic |
| Icons | lucide-react |
| Charts | recharts |
| Backend | Supabase (PostgreSQL + Auth + RLS + Edge Functions) |
| Deployment | Vercel (automatic CI/CD) |
| Tokenization | Dictionary-based max forward matching |
| Email | Brevo (Sendinblue) transactional emails |
