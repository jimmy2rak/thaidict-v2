# 2026-07-16 中泰词典工作日志

## 7. PyThaiNLP newmm 分词方案落地（commit 40f4680 已推）

### 需求
用户反馈例句分词仍大量出错/未分，截图中 `ของนาฬิกา`、`ควบคุมราคาสินค้า`、`มีประสิทธิภาพสูง` 等被过度合并。要求用 PyThaiNLP 的 newmm，并「把 newmm 直接导入数据库中」。

### 方案架构（三层）
1. **数据库金标准**：所有现有例句（dictionary / community_words / sentences）用 PyThaiNLP newmm 预分词，结果写入 `segmented` 字段；WordDetailPage 优先使用 `ex.segmented` 渲染，无需运行时计算。
2. **服务端实时兜底**：独立 Python 微服务 `services/thai-segment/` 运行 PyThaiNLP newmm；Next.js `/api/thai-segment` 优先调用它，服务不可用时降级为服务端 JS 分词器。
3. **新数据入库兜底**：`saveCommunityWord` / `addDictionaryWord` 在入库前调用 `fillSensesSegments`，对 AI 漏给 segmented 的例句自动补充分词。

### 新增文件
- `services/thai-segment/main.py`：FastAPI，含 `/segment`、`/batch`、`/legacy`、`/health`。
- `services/thai-segment/requirements.txt`：fastapi + uvicorn + pythainlp + pydantic + supabase + python-dotenv。
- `services/thai-segment/Dockerfile`：Python 3.11 slim 容器。
- `services/thai-segment/README.md`：部署与接口说明。
- `scripts/segment_existing_data.py`：批量回填数据库。环境变量 `SUPABASE_URL` + `SUPABASE_SERVICE_ROLE_KEY`；支持 `DRY_RUN=1`、`ONLY_EMPTY=1`（默认只填缺失）、`BATCH_SIZE`。
- `scripts/segment_one.py`：单句分词测试。
- `src/lib/segmentExamples.js`：`fillExampleSegment` / `fillSensesSegments` / `fillSenseSegments` 浏览器端 API 补全工具。

### 修改文件
- `app/api/thai-segment/route.js`：优先调用 `THAI_SEGMENT_SERVICE_URL`，失败回退 `serverTokenize`。
- `src/lib/db/community.js`：`saveCommunityWord` 先 `fillSensesSegments` 再入库。
- `src/lib/db/search.js`：`addDictionaryWord` 先 `fillSensesSegments` 再入库；顺便补了缺失的 `setGlobal` import。
- `src/lib/ai-proxy.js`：prompt 中强调 `segmented` 必须提供，并给出分词粒度示例。
- `.env.example`：增加 `THAI_SEGMENT_SERVICE_URL`。

### 本地限制
当前 sandbox 因内存/磁盘不足，无法安装 PyThaiNLP（pip 安装被 OOM kill）。因此未能在本地跑 `segment_one.py` 实测。但代码已按 PyThaiNLP 标准 API 设计，部署 Python 服务并设置环境变量后即可生效。

### 用户后续动作
1. 部署 `services/thai-segment/` 到任意 Python 容器（Railway/Render/Fly/自有服务器），记下公网 URL。
2. 在 Vercel Dashboard 添加环境变量 `THAI_SEGMENT_SERVICE_URL=https://your-service/segment`。
3. 在本地或服务器运行 `scripts/segment_existing_data.py` 回填旧数据（建议先 `DRY_RUN=1` 看日志）。
4. 之后新例句在审批/入库时会自动补全 segmented；AI prompt 也要求 AI 直接给出高质量 segmented。
